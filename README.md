# nourish

The reminder for nosh.jp

## Usage

- `NOURISH_EMAIL`
- `NOURISH_PASSWORD`
- `NOURISH_SLACK_URL`
  - [Incoming Webhook URL](https://api.slack.com/messaging/webhooks)
- `NOURISH_SLACK_CHANNEL`
  - A name of channel

Optional

- `NOURISH_REMIND_HOUR`
  - A threshold remind hour (default: 24)
